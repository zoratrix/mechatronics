- Мехатроника

Там же находится [презентация с основной организационной информацией](https://github.com/zoratrix/mechatronics/blob/main/org/0%20%D0%BE%D1%80%D0%B3%20%D0%BC%D0%BE%D0%BC%D0%B5%D0%BD%D1%82%D1%8B.pdf).
